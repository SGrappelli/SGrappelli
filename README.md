## Hi, I'm Konstantin 👋

Solo founder based in Batumi, Georgia. Building in public.

### 🚀 What I'm building

**[Pronto](https://trypronto.app)** — open-source POS, CRM, appointment booking 
and inventory for service businesses (salons, barbershops, auto repair shops).

- Self-hosted with one Docker command: `docker compose up -d`
- Omnichannel notifications: Email · Telegram · WhatsApp · Viber
- Clients book without creating an account — zero commission
- MIT license · Free forever for self-hosters

→ **[github.com/SGrappelli/pronto](https://github.com/SGrappelli/pronto)**  
→ **[trypronto.app](https://trypronto.app)**

### 🛠 Stack

Next.js 14 · Supabase · PostgreSQL · Tailwind CSS · Docker · Cloudflare R2
